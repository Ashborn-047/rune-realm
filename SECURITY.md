# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please do **not** open a public issue.

Instead, report it privately by emailing the maintainer with:

- A description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will acknowledge reports as quickly as possible and aim to provide a remediation timeline after validation.
