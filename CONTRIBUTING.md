# Contributing

Thanks for your interest in contributing to Rune Realm.

## Setup

1. Fork the repo
2. Clone your fork
3. Install dependencies:

```bash
npm install
```

4. Run locally:

```bash
npm run dev
```

## Guidelines

- Keep UI consistent with the existing runic aesthetic
- Prefer small, focused pull requests
- Test responsive behavior for mobile, tablet, and desktop
- Keep animations smooth and performance-conscious

## Pull Requests

- Use clear commit messages
- Describe behavior changes in the PR summary
- Include screenshots/GIFs for visual changes when possible
